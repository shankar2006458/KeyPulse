# KeyPulse

A futuristic real-time keyboard tester and RGB visualizer built with pure HTML, CSS, and JavaScript.

KeyPulse combines mechanical keyboard inspired sound effects, animated RGB lighting, live key analytics, and a modern cyberpunk UI into a smooth desktop experience.

## Features

* Real-time keyboard testing
* RGB reactive lighting effects
* Multiple RGB modes

  * Reactive
  * Wave
  * Rainbow
  * Breathing
* Multiple sound themes

  * Mechanical
  * Blue Switch
  * Thocky
  * Retro Keyboard
  * Typewriter
  * Laser
* Live keyboard statistics
* Key press history
* Active key detection
* Caps Lock / Num Lock / Scroll Lock indicators
* Heatmap glow effect
* Particle animated background
* Glassmorphism UI
* Fullscreen mode
* Desktop-only optimized experience
* Responsive animated dropdowns
* Smooth ripple and glow animations

## Built With

* HTML5
* CSS3
* Vanilla JavaScript
* Web Audio API

## Sound Themes

KeyPulse includes multiple keyboard-inspired sound profiles powered by the Web Audio API.

| Theme       | Description                 |
| ----------- | --------------------------- |
| Mechanical  | Classic keyboard click      |
| Blue Switch | Sharp tactile sound         |
| Thocky      | Deep premium keyboard sound |
| Retro       | Old-school terminal feel    |
| Typewriter  | Vintage typewriter effect   |
| Laser       | Futuristic sci-fi sound     |

## RGB Modes

| Mode      | Effect                       |
| --------- | ---------------------------- |
| Reactive  | Keys glow when pressed       |
| Wave      | Continuous RGB animation     |
| Rainbow   | Multicolor gradient lighting |
| Breathing | Pulsing glow effect          |


## Performance Notes

KeyPulse is optimized for:

* Desktop keyboards
* Full-size screens
* Modern Chromium-based browsers

Mobile devices are intentionally blocked for the best experience.
